# Modelagem de Ameaças Utilizando Inteligência Artificial

## Contexto

A **FIAP Software Security**, empresa especializada em Segurança de Sistemas, está analisando a viabilidade de uma nova funcionalidade para otimizar seu software de análise de vulnerabilidades em arquiteturas de sistemas.

O objetivo da empresa é utilizar **Inteligência Artificial** para identificar e tratar automaticamente vulnerabilidades que possam colocar em risco a segurança dos sistemas desenvolvidos por arquitetos(as) e desenvolvedores(as).

Um dos principais desafios é empregar IA para realizar **modelagem automática de ameaças**, baseada na metodologia **STRIDE**, a partir de **diagramas de arquitetura de software em formato de imagem**.

Para validar a viabilidade dessa funcionalidade, será desenvolvido um **MVP (Minimum Viable Product)** focado na **detecção supervisionada de ameaças**.

---

## Objetivos

O projeto tem como objetivos principais:

- Desenvolver uma IA capaz de interpretar automaticamente diagramas de arquitetura de sistemas;
- Identificar componentes de arquitetura, tais como:
  - Usuários
  - Servidores
  - Bases de dados
  - APIs
  - Outros elementos arquiteturais;
- Gerar automaticamente um **Relatório de Modelagem de Ameaças**, baseado na metodologia **STRIDE**;
- Construir ou buscar um **dataset contendo imagens de arquiteturas de software**;
- Anotar o dataset para treinamento supervisionado do modelo, permitindo:
  - Reconhecimento dos diferentes componentes arquiteturais;
- Treinar o modelo de IA;
- Desenvolver um sistema capaz de:
  - Associar vulnerabilidades a cada componente identificado;
  - Apontar contramedidas específicas para cada tipo de ameaça.
